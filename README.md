Este proyecto consiste en una aplicación web para la gestión de una biblioteca, permitiendo administrar Autor@es y sus respectivos Libros. Ha sido desarrollado siguiendo una estructura API REST para el backend junto con una interfaz dinámica en el frontend.

Se ha usado DB Browser SQlite como aplicación para la creación de la base de datos, Postman para las pruebas de los GET, POST, PUT y DELETE en el backend antes de desarrollar el frontend y GitHub/Git para el almacenamiento del proyecto. Del mismo modo, se ha usado en el backend Node.js, Express, SQLite3 y Knex.js. En el caso del frontend se ha usado HTML, CSS, JS, Axios y Parcel.

Para la puesta en marcha del proyecto se deberá tener instalado Node.js y se descargará el proyecto del repositorio en el que se encuentra y se descomprimirá. Una vez se haya hecho esto se abrirá el IDE Visual Studio Code y se procederá, con el explorador de archivos de VSC, a abrir la carpeta del Proyecto. Una vez hecho esto se abrirán 2 terminales en VSC, una se ubicará en la carpeta de frontend y otra en la de backend, y una vez ubicados lanzaremos los comandos "npm install" en ambas para descargar e instalar las diferente dependencias (teniendo previamente la aplicación de Node.js descargada e instalada). Una vez finalizados estos pasos se podrán usar los comandos "npm start" en ambas terminales para lanzar el proyecto, y se usará el localhost:1234/index.html para ir a la página principal del proyecto, donde se podrá trabajar con la Biblioteca.
